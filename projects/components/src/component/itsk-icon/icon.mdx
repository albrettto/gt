# Добавление файла с иконками в проект
В файле `angular.json` в секцию assets добавьте файлы из пакета itsk-ng-lib

```json
"assets": [
    "src/favicon.ico",
    {
        "glob": "**/*",
        "input": "path-to-node-modules/@grafit/angular/assets",
        "output": "/assets/"
    }
```

# Добавление спрайта с иконками в документ
В `app.component` добавьте сервис работы с иконками, и включите спрайт в документ

```javascript
constructor(private iconsService: ItskIconService) {
    this.iconsService.addSprite('assets/icon.svg');
  }
```

# Импорт модуля

```javascript
@NgModule({
  declarations: [],
  exports: [],
  imports: [
    ItskIconModule
  ]
})
export class YourModule {
}
```

# Использование иконок в шаблоне

```html
    <itsk-icon [name]="'icon-name'"></itsk-icon>
```
